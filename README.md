# Yali Katz's Portfolio Website

This is the official repository for Yali Katz's personal portfolio website built with modern web standards.

## Project Structure

```
/
├── index.html # Home page
├── assets/
│   ├── css/ # Stylesheets
│   │   ├── main.css # Main styles
│   │   ├── responsive.css # Responsive styles
│   │   ├── rtl.css # RTL support for Hebrew content
│   │   ├── home-styles.css # Home page styles
│   │   ├── button-animation.css # Button animations
│   │   └── qa.css # Q&A styles
│   ├── js/ # JavaScript files
│   │   └── main.js # Main JavaScript functionality
│   └── images/ # Image assets
├── components/ # Reusable HTML components
│   └── footer.html # Site footer
├── pages/
│   ├── projects/ # Projects section
│   │   ├── index.html # Projects page
│   │   └── projects-styles.css # Projects styles
│   └── work-experience/ # Work experience section
│       ├── index.html # Work experience page
│       └── experience-styles.css # Work experience styles
├── Personal-Projects/ # Personal projects page
│   └── personal-projects.html
├── Work-Experience/ # Work experience page
│   └── work-experience.html
├── styles.css # Global styles
└── heb_style.css # Hebrew-specific styles
```

## Tech Stack

- HTML5
- CSS3 with CSS variables for theming
- Vanilla JavaScript
- Responsive design with a mobile-first approach
- RTL support for Hebrew content

## Features

- Clean, modern portfolio layout
- Responsive design for desktop and mobile
- Bilingual support for English and Hebrew content
- Separate project and work experience pages
- Modular CSS structure for easier maintenance
- Fast-loading static site architecture

## Development

To run this website locally:

1. Clone this repository
2. Open `index.html` in your browser

## Deployment

This website is suitable for static hosting and can be deployed using GitHub Pages.
